# DBX 项目分析文档

## 项目概述

DBX 是一个轻量级、高性能的开源数据库管理工具，支持 60+ 种数据库，体积仅约 20MB。项目采用现代化的技术栈，提供桌面应用、Docker 自托管和 Web 版本三种部署方式。

**核心特性：**
- 零运行时依赖（无需 Java、Python 等）
- 内置 AI SQL 助手和 MCP 协议支持
- 支持 MySQL、PostgreSQL、SQLite、Redis、MongoDB、ClickHouse 等 60+ 数据库
- 提供桌面应用（macOS/Windows/Linux）、Docker 和 Web 版本
- 代码体积小巧（~20MB），启动快速

**版本信息：**
- 当前版本：0.5.56
- 许可证：Apache-2.0
- 官网：https://dbxio.com

## 项目架构

### 整体架构

DBX 采用 **Monorepo** 架构，使用 pnpm workspace 管理多个包：

```
dbx/
├── src-tauri/          # Tauri 桌面应用入口
├── crates/             # Rust 核心库
│   ├── dbx-core/       # 核心业务逻辑
│   └── dbx-web/        # Web 版本后端
├── apps/
│   └── desktop/        # Vue 3 前端应用
├── packages/           # Node.js 工具包
│   ├── node-core/      # 共享工具库
│   ├── cli/            # CLI 工具
│   └── mcp-server/     # MCP 服务器
├── agents/             # JDBC 代理驱动（Java）
└── docs/               # 文档站点
```

### 核心模块详解

#### 1. src-tauri（桌面应用入口）

**职责：** Tauri 应用入口，处理窗口管理、系统托盘、深度链接等桌面特性

**关键文件：**
- `main.rs` - 应用入口
- `lib.rs` (1364 行) - Tauri 命令注册和初始化
- `data_dir.rs` - 数据目录管理
- `macos_app_delegate.rs` - macOS 特定功能
- `window_state_guard.rs` - 窗口状态管理

**依赖关系：**
- 依赖 `dbx-core` 核心库
- 通过 Tauri 命令与前端通信

#### 2. crates/dbx-core（核心业务逻辑）

**职责：** 所有数据库操作的核心实现，包含 60+ 个模块

**核心模块分类：**

**连接管理：**
- `connection.rs` (5151 行) - 数据库连接管理
- `connection_secrets.rs` (1145 行) - 连接密钥加密存储
- `ssh_config.rs` - SSH 隧道配置

**查询执行：**
- `query.rs` (4317 行) - 查询执行引擎
- `query_execution_sql.rs` - SQL 执行逻辑
- `query_cancel.rs` - 查询取消
- `query_result_export.rs` - 结果导出

**Schema 管理：**
- `schema.rs` (6306 行) - Schema 浏览器
- `schema_diff.rs` (2140 行) - Schema 对比
- `table_structure_sql.rs` - 表结构编辑

**数据操作：**
- `data_grid_sql.rs` (4533 行) - 数据网格
- `table_import.rs` (3151 行) - 表导入
- `table_export.rs` - 表导出
- `transfer.rs` (6650 行) - 数据传输
- `data_compare.rs` - 数据对比

**AI 功能：**
- `ai.rs` (3338 行) - AI SQL 助手
- `ai_cli_agent.rs` - CLI AI 代理
- `ai_codex_cli.rs` - Codex CLI 集成

**JDBC 代理系统：**
- `agent_manager.rs` - 代理管理器
- `agent_connection.rs` - 代理连接
- `agent_loop.rs` - 代理事件循环
- `agent_service.rs` - 代理服务
- `agent_catalog.rs` - 代理目录
- `jdbc.rs` - JDBC 桥接

**特定数据库支持：**
- `redis_ops.rs` - Redis 操作
- `mongo_ops.rs` - MongoDB 操作
- `mq/` - 消息队列管理（Kafka、Pulsar、RocketMQ）

**导出功能：**
- `csv_export.rs` - CSV 导出
- `xlsx_export.rs` - Excel 导出
- `text_export.rs` - 文本导出
- `query_result_export.rs` - 查询结果导出

**安全特性：**
- `production_safety.rs` (688 行) - 生产环境安全检查
- `sql_risk.rs` - SQL 风险评估
- `sql_editability.rs` - SQL 可编辑性检查

**存储与配置：**
- `storage.rs` (3276 行) - 本地存储管理
- `cloud_sync.rs` - 云同步

#### 3. crates/dbx-web（Web 版本后端）

**职责：** 提供 Web 版本的 HTTP API 服务

**技术栈：**
- Axum - Web 框架
- Tower HTTP - 中间件（CORS、压缩、日志）
- WebSocket 支持

**关键特性：**
- 用户认证（Argon2 密码哈希）
- AES-GCM 加密
- 与 dbx-core 共享业务逻辑

#### 4. apps/desktop（前端应用）

**职责：** 用户界面，基于 Vue 3 + TypeScript

**目录结构：**
```

│   ├── connection/    # 连接管理

│   ├── admin/         # 数据库管理

├── stores/            # Pinia 状态管理
├── utils/             # 工具函数
```

**核心组件：**
- `App.vue` (2205 行) - 主应用组件
- `QueryEditor.vue` (3685 行) - SQL 查询编辑器
- `ConnectionDialog.vue` (5980 行) - 连接对话框
- `AiAssistant.vue` (2299 行) - AI 助手界面

**UI 特性：**
- 深色模式
- 9 种编辑器主题

- 虚拟滚动（大数据集）
- 响应式布局

#### 5. packages/（Node.js 工具包）

**@dbx-app/node-core**
- 共享的数据库连接工具
- 支持 MySQL、PostgreSQL、SQLite、Redis

**@dbx-app/cli**

**@dbx-app/mcp-server**
- Model Context Protocol 服务器
- 允许 AI 代理（Claude Code、Cursor 等）查询数据库
- 复用 DBX 已配置的连接

#### 6. agents/（JDBC 代理驱动）

**职责：** 通过 Java JDBC 扩展数据库支持

**技术栈：**
- Java + Gradle
- 35+ 个数据库驱动模块

**支持的数据库：**

## 部署方式

### 1. 桌面应用

**支持平台：**
- macOS（Homebrew）
- Windows（Scoop、WinGet、NSIS 安装包）
- Linux（Flatpak）

**安装示例：**
```bash
# macOS
brew install --cask dbx

# Windows (Scoop)
scoop bucket add dbx https://github.com/t8y2/scoop-bucket
scoop install dbx

# Windows (WinGet)
winget install t8y2.dbx

# Linux (Flatpak)
flatpak remote-add --if-not-exists flatpark https://dl.flatpark.org/flatpark.flatpakrepo
flatpak install flatpark com.dbxio.dbx
```

**特性：**
- 单文件安装
- 自动更新
- 系统托盘集成
- 深度链接（dbx:// 协议）
- 文件关联（.sql、.db、.sqlite、.duckdb）

### 2. Docker 自托管

**快速启动：**
```bash
docker run -d --pull=always --name dbx -p 4224:4224 -v dbx-data:/app/data t8y2/dbx:latest
```

**Docker Compose：**
```yaml
services:
  dbx:
    image: t8y2/dbx:latest
    pull_policy: always
    ports:
      - "4224:4224"
    volumes:
      - dbx-data:/app/data
    restart: unless-stopped
```

**特性：**
- 多架构支持（amd64/arm64）
- 数据持久化
- 反向代理支持（可配置 base path）
- 中国镜像加速（CNB）

### 3. Web 版本

**技术栈：**
- 前端：Vue 3 + Vite
- 后端：Axum（Rust）
- 通信：WebSocket + HTTP API

**特性：**
- 与桌面版相同的功能
- 浏览器访问
- 团队共享
